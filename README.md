# What is OpsPilot?

OpsPilot is an **AI-powered SaaS ERP platform** built using **Python Flask**.

It combines multiple business tools into one modern application:

- CRM
- HRMS
- Employee Management
- Finance
- Inventory
- AI Assistant
- Task Management
- Analytics Dashboard
- Business Automation

Instead of purchasing many different software products, businesses can run everything from one dashboard.

# Installation

```bash
git clone https://github.com/YOUR_USERNAME/opspilot.git

cd opspilot

pip install -r requirements.txt

python app.py
```

Open

```
http://localhost:8000
```
